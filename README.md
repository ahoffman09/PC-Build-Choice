# PC Build Choices
This project was my first-ever web design project, done using HTML and CSS. This was my first ever hands-on experience with web design, and I did it as I was teaching myself the two languages. This is technically a survey - it gathers what the person taking the survey used for each component of their pc, like CPU and Motherboard. I chose some interesting stylistic designs, not ones that I would necessarily choose in this strong of contrast but still a scheme I would make nonetheless. This is my very first ever actual design project, so it was not made with many of the style and ease on the eyes choices I would have made today. It also plays around with the different styles of forms you can make, which gave me experience with how to make them in HTML/CSS.
